# Build your Backend using FastAPI

## Pre-requisites
* Python (3.6+)

## Running the Code

1. Clone the project
   * `git clone git@github.com:ashmibanerjee/fastapi-backend.git`
   * `cd fastapi-backend`
2. Create virtual environment
   * `virtualenv venv`
   * `source venv/bin/activate`
3. Install dependencies 
   * `pip3 install -r requirements.txt`
4. Run the server
   * `python3 src/main.py`
